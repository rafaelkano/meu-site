O que é HTML e de onde veio

Um breve histórico do desenvolvimento do HTML e CSS para entender o porque de sua criação e de seu estado atual (senta que lá vem história).

O que é o HTML? A sigla significa HiperText Markup Language, ou linguagem de marcação de hipertexto. E o que isso significa? Que o HTML permite estruturar uma página Web através de um documento de texto, utilizando marcações (TAGs) que dão significado a cada parte do documento.
